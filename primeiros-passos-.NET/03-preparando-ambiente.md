# Preparando o ambiente

## Instalando .NET

Faça o download do .NET [clicando aqui](https://dotnet.microsoft.com/en-us/download). Atente-se para baixar a versão SDK (software development kit).
Ao término do download faça a instalação do arquivo baixado.

Para verificar se o .NET está instalado abra o prompt de comando/terminal e digite: 

	dotnet --version

o terminal retornará a versão instalada em seu computador.

## Instalando Visual Studio Code

Faça o download do Visual Studio Code [clicando aqui](https://code.visualstudio.com/).
Ao término do download faça a instalação do arquivo baixado.

No Visual Studio Code instale as extensões C# e C# extensions
